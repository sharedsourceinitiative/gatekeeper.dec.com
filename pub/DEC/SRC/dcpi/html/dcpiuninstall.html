dcpiuninstall(1)

NAME

dcpiuninstall - Uninstall DIGITAL Continuous Profiling Infrastructure files

SYNOPSIS

dcpiuninstall

DESCRIPTION

Dcpiuninstall is a shell script that is generated when the DCPI system is installed. When run, Dcpiuninstall removes DCPI binaries and man pages, and replaces /sbin/loader with the original version that was saved during the install process. Dcpiuninstall does not remove profile databases or ancillary files that may remain from running dcpid.
