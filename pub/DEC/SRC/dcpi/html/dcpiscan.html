dcpiscan(1)

NAME

dcpiscan - Generate an image map used by dcpid to find images

SYNOPSIS

dcpiscan [file|directory...]

FLAGS

None.

DESCRIPTION

Dcpiscan searches for images (executables and shared libraries) in the specified directories and files, and prints to standard output an image map for these images. This image map should be saved into a file and then fed into dcpid(1) via the -m option to allow dcpid to find the names of images it encounters while profiling the system.
