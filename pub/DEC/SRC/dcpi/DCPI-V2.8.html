DCPI-V2.8 Release Notes 
Changes since release V2.7:

- New loader for 4.0c and higher.
  Dynamically linked images can be profiled on machines running
  DU V3.2 or V4.0*, but not V5.0*
DCPI-V2.7 Release Notes 
Changes since release V2.6:

- Removed loader that doesn't work with 4.0d
  Dynamically linked images can be profiled only on machines running
  DU V3.2, V4.0 (Rev 386), and V4.0B (Rev 564).
DCPI-V2.6 Release Notes 
Changes since release V2.3:

- Major update to dcpi2pix to generate more accurate data.

- Added loader that should work with 4.0d

- Added support for new 21264 instructions.
