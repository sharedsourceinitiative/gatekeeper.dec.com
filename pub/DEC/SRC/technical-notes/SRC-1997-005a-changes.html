Changes included in SRC Technical Note 1997-005a

The following list reflects the complete set of changes made to SRC Technical Note 1997-005 to produce the revised note 1997-005a.

  • An extra fp_contents argument has been added to the _run_tool primitive. This argument allows the caller to specify that the fingerprints of derived files should be computed from the contents of the files. By default, a unique fingerprint is chosen non-deterministically for each newly-created derived file.

  • The way in which deleted files are handled in _run_tool results has been changed. Deleted files are no longer removed from the resulting binding, but are instead bound to the value false.

  • New _sub functions have been defined on lists and bindings. For consistency, the _subtext primitive has been renamed _sub. Also, _sub has been added to the list of reserved identifiers, and _subtext has been removed from that list.

  • The name _map has been added to the list of reserved identifiers.

  • A mistake in the grammar has been fixed: a hexadecimal Integer is allowed to contain any number of hex digits, not at most two as in the original grammar.

  • Several minor typos have been corrected.
