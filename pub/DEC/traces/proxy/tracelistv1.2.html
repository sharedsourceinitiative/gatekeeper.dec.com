Digital's Web Proxy Traces: Currently Available Traces V1.2 format

The trace files listed here represent the same trace as do the V1.0 format trace files, but with a few additional fields, as described in the documentation.

The current traces contain data taken between 29 August 1996 and 22 September 96. (Some traces may not cover entire days.) This is a total of approximately 24,477,674 references.

For a subset of the references including only those references from clients that were within a short distance of the proxy where the trace was made, see the subset traces.

The "headers" file for each trace contains statistics about the trace, including the number of references.

In addition, we provide files mapping between the unique IDs (UIDs) in these traces, and keyed-MD5 signatures of the orginal fields. These mappings may be useful, in the future, for correlating between traces from different sites.

  • Map between between keyed-MD5 fingerprint of client addresses and UIDs (384 KBytes, 17354 entries)
  • Map between between keyed-MD5 fingerprint of server names and UIDs (3048 KBytes, 140506 entries)
  • Map between between keyed-MD5 fingerprint of URL paths and UIDs (85232 KBytes, 3934419 entries)
  • Map between between keyed-MD5 fingerprint of URL queries and UIDs (18544 KBytes, 855986 entries)
Note: some of these mapping files are quite large! Also note that these are the same mapping files used for the V1.0 format trace files.

The complete set of traces is about 602 MBytes; the complete set of mapping files is about 107 MBytes.

We will not release the original mapping between these fields and keyed-MD5 signatures, nor will we release the MD5 key. Please do not ask.
