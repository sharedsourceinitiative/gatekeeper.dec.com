This directory contains archive copies of the most recent Lynx development source code. We are currently working on Lynx 2.8.6.

Mirror site(s):
The primary HTTP distribution site is http://lynx.isc.org/current/;
The primary FTP distribution site is ftp://lynx.isc.org/lynx/current/; also known as ftp://ftp.us.browser.org/lynx/current/. Lynx users may use the index page on the FTP site for an organized view of the current distribution.
Mirrors may be found on:
ftp://ftp.mirror.ac.uk/sites/lynx.isc.org/lynx/
ftp://gd.tuwien.ac.at/infosys/browsers/lynx/
ftp://ftp.eu.uu.net/pub/unix/www/lynx/
http://open-systems.ufl.edu/mirrors/ftp.isc.org/lynx/
ftp://sunsite.cnlab-switch.ch/mirror/lynx/
ftp://ftp.cyf-kr.edu.pl/pub/unix/lynx/
ftp://ftp.netlab.is.tsukuba.ac.jp/pub/network/WWW/lynx/
ftp://ftp.u-aizu.ac.jp/pub/net/www/lynx/
For the latest stable release and other help, see http://lynx.isc.org/. For DOS386/Win32 binaries, see below.
The development version also has a breakout subdirectory.
