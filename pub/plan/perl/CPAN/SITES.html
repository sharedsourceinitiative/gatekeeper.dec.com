SITES

Network

Welcome to the CPAN, the Comprehensive Perl Archive Network. This document describes the CPAN itself, the Mirror Network.

The master CPAN site is hosted by FUNET, the Finnish University NETwork (Espoo, Finland, Europe).

Select the site nearest to you from the list below, 'Registered CPAN sites' or from our mirror database to get the best response time and bandwidth. NOTE: one way to select 'the nearest' is to use the CPAN Multiplexer which can be used either to guess or to select the nearest site for you based on your host's Internet domain name.

The list of the registered CPAN sites follows. Please note that the sorting order is alphabetical on fields:


Continent

   |-->Country
         |
         |-->[state/province]

How You Can Help

Inform

If you are aware of Perl resources that are not in the CPAN (you did check the contents listings in indices/, didn't you?) please tell us. We will either add the resource to the local mirror, arrange for regular mirroring of the content or provide a link depending on the size, value and stability.

Contribute

If you have modules/scripts/documentation yourself that you would like to contribute to CPAN, please read the file modules/04pause.html and let us know.

Register

If you have a reliable and well-connected ftp or http site where you can keep CPAN (about 850 MB currently), more mirror sites are welcome. Feel free to volunteer and contact the CPAN administrators describing your mirroring setup, please see the top of the file MIRRORED.BY or register it on-line. Of course, first check (see from the above list) whether your area (network-connectivity-and-bandwidth-wise) is already well covered. Redundancy and fault-tolerance are good things for ftp archives like CPAN but virtues can be exaggerated. Mirroring software is available from the CPAN or from the Mirror home page which will have the most recent version. We also encourage the use of rsync.

Private/Local Mirroring

If you want to configure a private/local mirror of CPAN but do not want to advertise it you do not need to contact the CPAN admistrators. You need to assess which of the public CPAN sites would give the best ftp bandwidth for you (at off-peak hours) and contact the mirror admin via the email address given in the MIRRORED.BY file to learn the best time of the day to do your mirror. Mirroring software is available from the CPAN or from the Mirror home page which will have the most recent version. We also encourage the use of rsync.

We encourage you to read our FAQ Section on how to mirror CPAN and visit the mirror database to find a mirror close to you and to register yours with CPAN.
